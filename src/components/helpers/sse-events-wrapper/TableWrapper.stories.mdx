# SSE Table implementation
<hr />

This is an example implementation of the SSEEventsWrapper. In this case we're showcasing a
table that gets mutated by sse events. First it fetchs the initial data from an endpoint
using the **`useAutoFetch`** helper, then starts getting the `delta` through sse. These
events will cycle from update, create, remove if **Auto Pilot** is selected.

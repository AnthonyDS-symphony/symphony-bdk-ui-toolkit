# Phone Input Field
<hr />

This is an International Phone number Input, with the ability to choose a particular
country from a list, providing a country code for it.
It wraps this component [react-phone-input-2](https://github.com/bl00mber/react-phone-input-2)

<Preview withSource="open">
  <Story name="regular">
    <PhoneInputField value="+1" label="Default Phone Input field"/>
  </Story>
</Preview>

### Minimum Requirements
In order to make this component work properly, you must import the dist.css file from the Toolkit
like so:

```jsx
import '../node_modules/symphony-bdk-ui-toolkit/dist/index.css';

```

Please for further customization please follow the above documentation,
everything passed to PhoneInputField will be propagated and **override** any prop.

# Progress Indicator

Progress Indicator component, showing in which step a certain multi-step action is.

Simply takes in the number of steps, and which step it's currently on. Can be set horizontally as well.
The length of the Progress Indicator can also be changed, by passing the ```length``` value, as a number prop (in px).
It is a controlled component therefore, a wrapper will most likely be used.


```jsx
export const ProgressController = (props) => {
  const { numberOfSteps, ...rest } = props;
  const [current, setCurrent] = useState(0);
  return (
    <Box justify="center" align="center">
      <Box horizontal>
        <ProgressIndicator
          numberOfSteps={numberOfSteps}
          currentStep={current}
          {...rest}
        />
      </Box>
      <Box horizontal>
        <Button
          size="small"
          onClick={() => {
            if (current > 0) {
              setCurrent(current - 1);
            }
          }}
        >{'<'}
        </Button>
        <Button
          size="small"
          onClick={() => {
            if (current < numberOfSteps - 1) {
              setCurrent(current + 1);
            }
          }}
        >{'>'}
        </Button>
      </Box>
    </Box>
  );
};
```

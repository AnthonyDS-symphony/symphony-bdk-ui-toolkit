# Quote Card
<hr />

A card for displaying quotes. It takes producs, and descriptions as options. It implements
a context menu for actions.

### Simple

This is a basic card witch contrasts dealer/buyer pricing. In order to have that
you'll have to provide an property called **`panelData`**. here's a sample of that structure

```jsx
const panelData = {
  dealerName: 'Dealer two',
  dealerPayedValue: 1.75,
  clientName: 'Client one',
  clientPayedValue: 3,
};
```

### Product Data

This card shows the keywords that define a particular quote. In order to have that
you'll have to provide an property called **`productData`**. here's a sample of that structure

```jsx
const productData = {
  name: 'IRS',
  currency: 'USD',
  rateIndex: '3M-LIBOR',
  clearingHouse: 'EUREX',
  startDate: 'spot',
  tenorDate: '1y',
  roll: 'IMM',
  size: {
    type: 'DV01', currency: 'USD', value: '3', multiplier: 'k',
  },
  payDirection: 'pay',
};
```

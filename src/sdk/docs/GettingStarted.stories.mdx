  The **Symphony Developer Framework** was built to help developers and companies to accelerate their business efforts.
  It comprehends a set of tools that are tailored made for companies that need to integrate Symphony with their existing
  workflows, or to create brand new applications that leverages Symphony's Data Security model. It provides:
</div>

- **[CLI](?path=/docs/developer-framework-cli--page)** A command line tool that allows you to in a manner of seconds have a functional Bot project up and running
  in your machine, also, to have an complex extension app boilerplate with themable capabilities, that already has
  all authentication, bootstrap and symphony integration done for you, and many more.
- **[Bot Development Framework](?path=/docs/developer-framework-bot-sdk--page)** Bot Development framework, which ships with authentication support, command matchers, REST api capabilities, and many more.
  Its foundation is built on top of best development practices and Financial companies in heart.

- **[Toolbox UI](?path=/docs/developer-framework-toolbox-ui--page)** A Component library, that helps you to build complex applications rapidly. It is meant to reduce
    the development of Extension apps from months to weeks. It has from Layout, to forms, and even complex
    financial charts, to allow you get the best out of your workflow, or business requirements.

## Description
This tool generates project scaffolds for *Bot* and *Extension Application* projects,
including RSA key generation and installing dependencies. *Bot* and *Extension
Application* projects generated using the CLI are designed to integrate with each other
to form complete solutions that deliver the best user experience.

## Requirements
* NodeJS
* JDK 8
* Maven 3
* Git

## Installation
For npm:
```bash
npm i -g symphony-bdk-cli
```

For yarn:
```bash
yarn global add symphony-bdk-cli
```

### Dependencies
You can check if any dependencies are unmet by running a diagnostic on your environment
using the `--check-deps` argument.
```bash
symphony-bdk-cli --check-deps
```

## Commands
* ```symphony-bdk-cli --check-deps``` checks that all required dependencies are met
* ```symphony-bdk-cli --bot``` creates a new bot project
* ```symphony-bdk-cli --app``` creates a new extension app project
* ```--run``` can be appended to `--app` or `--bot` to generate a project then run it immediately
* ```symphony-bdk-cli --bot command-handler``` creates a new command handler in a existing bot project
* ```symphony-bdk-cli --app message-template``` creates a new notification in a existing extension app project
* ```symphony-bdk-cli --toolkit``` launches the component library browser
* ```--gen-certs``` generates RSA key pairs and jwt tokens
